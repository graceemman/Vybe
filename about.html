About VYBE

A compact web project combining music, mood, utilities, and a polished visual style.

Project Report

Overview

VYBE is a multi-page static website built using HTML, CSS, Bootstrap, JavaScript, and jQuery. The homepage presents a premium glassmorphism interface with an animated gradient background, live date and time, a smooth vibe color changer, and an embedded HTML5 music player.

The tools page demonstrates practical JavaScript features including mood recommendations, task management, number sorting, and a jQuery-powered toggle animation. The contact page includes basic form validation, while this page documents the concept and team contribution.

Team Contribution

Responsibilities

  • UI Design: Modern layout, spacing, colors, and glassmorphism style.
  • Frontend Development: HTML page structure and Bootstrap grid system.
  • Interactivity: JavaScript tools, live clock, and form validation.
  • Testing: Checked navigation, responsive layout, and user inputs.
